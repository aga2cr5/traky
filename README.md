# traky
täällä on tallennettuna kaikki suoritetut ja palautetut kurssiharjoitukset
tietorakenteet ja algoritmit kurssilta. kaikki kirjoitettu koodi ei ole omaani vaan osittain valmiisiin koodeihin on lisätty/täydennetty/korjattu puuttuvia osia. jokaisen tehtävän tests.py tiedosto on ollut käytännössä valmiina kurssitehtävien testaamista varten. muutamaan taisin lisätä itse jotain testejä auttamaan tehtävien ratkaisussa. tehtäviä voi käyttää apuna tietorakenteet ja algoritmit kurssilla, mutta niitä ei tietystikään saa plagioida. käyttäjä on itse vastuussa mahdollisista plagiointisyytteistä.
